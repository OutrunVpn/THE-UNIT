# The UNIT Game Project

## Project Overview
The UNIT is a horror game designed with an aesthetic reminiscent of PlayStation 1 titles. Players navigate through a hauntingly immersive experience filled with chilling surprises and gameplay mechanics that keep them on edge.

## Features
- **PS1 Horror Aesthetic**: Emulating the distinctive style of 90s horror games, The UNIT creates a nostalgic yet terrifying atmosphere.
- **Flashlight Mechanic**: Navigate the dark corners of the game with a flashlight that illuminates key elements and reveals hidden threats.
- **Sanity System**: Players must manage their character's sanity as they face various horrors, affecting gameplay and interactions.
- **Looping Apartment**: Explore an enigmatic apartment that loops back on itself, creating a disorienting experience.

## Technical Stack
- **Three.js**: Used for 3D rendering and creating immersive environments.
- **HTML5**: For structuring the game’s interface and gameplay dynamics.

## Platform
- **Mobile Browser**: Optimized for a seamless experience on mobile devices.

## Installation Instructions
1. Clone the repository: `git clone https://github.com/OutrunVpn/THE-UNIT.git`
2. Open the `index.html` file in a mobile browser to start playing.

## Gameplay Controls
- **Swipe**: Move the character around.
- **Tap**: Interact with objects within the environment.
- **Hold**: Activate the flashlight and manage the sanity meter as necessary.
